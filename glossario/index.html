Glossário

Aqui estão algumas tags HTML e suas definições.

<blockquote>
Usamos esta tag para realizar citações em bloco.
<abbr>
Usamos esta tag indicar uma abreviação, podendo indicar seu significado para quando alguém passar o mouse por cima.
<strong>
Usamos esta tag indicar de forma semântica que o texto merece destaque.
<div>
Usamos esta tag para criar uma seção dentro do documento HTML, podendo inserir textos, imagens, links ou mesmo formatá-la da forma que preferir.
